## 混淆规律

1. **单字母混淆**: 抖音 v34.6.0 使用单字母作为混淆后的类名
2. **方法名混淆**: 方法名使用有意义的名称 (如 handleDiggClick, share)
3. **包结构保留**: 包结构保持不变 (com.ss.android.ugc.aweme.*)
4. **字段名不变**: 字段名和变量名保持不变 (如 play_addr, video_url)

## 注意事项

- 混淆类名在每个版本更新后可能会变化
- 使用运行时搜索 (DouyinFinder) 避免依赖混淆类名
- 系统 API 类名 (如 MediaPlayer, ImageView) 永不混淆
